# 远程连接使用指南

本文档提供了如何通过前端应用连接到远程腾讯云服务器的详细步骤和说明。

## 前置条件

- 确保您的设备可以访问互联网
- 现代浏览器（推荐Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）

## 远程服务器信息

**服务器地址**: 101.42.52.224:5000

## 访问方法

### 方法一：直接打开HTML文件（推荐）

1. 导航到项目目录 `eeg-analysis-frontend`
2. 找到并双击 `index.html` 文件
3. 浏览器会自动打开应用，并且会自动尝试连接到远程服务器
4. 查看页面右上角的提示信息，确认是否成功连接到远程服务器

### 方法二：使用静态服务器启动（适合开发环境）

如果您希望通过HTTP服务器方式启动前端（可能需要解决某些浏览器的安全策略限制）：

#### 使用Python启动

```bash
# 在 eeg-analysis-frontend 目录下执行
cd d:\新干电极头环上位机\只有ssvep\class_ssvep_control_tello\eeg-analysis-frontend
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000

#### 使用Node.js启动

```bash
# 先安装 http-server（如果尚未安装）
npm install -g http-server

# 在 eeg-analysis-frontend 目录下执行
cd d:\新干电极头环上位机\只有ssvep\class_ssvep_control_tello\eeg-analysis-frontend
http-server
```

然后在浏览器中访问：http://localhost:8080（或控制台显示的端口）

## 连接验证

1. 应用启动后，会自动进行健康检查，尝试连接到远程服务器
2. 您将在页面右上角看到连接状态提示：
   - ✅ 连接成功：显示绿色提示 "已成功连接到远程服务器"
   - ❌ 连接失败：显示红色提示 "连接远程服务器失败，请检查网络连接"

## 功能使用

成功连接到远程服务器后，您可以：

1. **上传脑电数据文件**：支持CSV、TXT、EDF格式
2. **进行数据分析**：通过远程API处理数据
3. **查看分析结果**：包括脑电波形、功率谱密度和脑波能量分布

## 常见问题排查

### 连接失败怎么办？

1. 检查您的网络连接是否正常
2. 确认您可以访问腾讯云服务器：在浏览器中访问 http://101.42.52.224:5000/health
3. 如果仍然无法连接，请检查是否有防火墙或网络限制阻止了连接
4. 查看浏览器控制台（按F12）获取详细错误信息

### 上传文件失败怎么办？

1. 确保文件格式正确（CSV、TXT或EDF）
2. 检查文件大小是否过大
3. 查看浏览器控制台获取详细错误信息

## 注意事项

1. 前端应用已经配置为自动连接到远程腾讯云服务器，无需手动修改任何配置
2. API配置文件位于 `src/services/api.js`，如需更改服务器地址可以修改此文件中的 `API_BASE_URL`
3. 远程服务器必须处于运行状态，前端才能正常连接

## 技术说明

- 前端使用纯HTML5 + CSS3 + JavaScript开发
- 所有依赖库通过CDN引入，无需额外安装
- 使用Fetch API与后端服务器进行通信
- 支持健康检查、数据上传和结果获取等API调用
